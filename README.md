# MyeongSeokKo Dev Log

TIL과 WIL을 기록하는 GitHub Pages + Jekyll 블로그입니다.

## 글 작성

새 글은 `_posts` 폴더에 `YYYY-MM-DD-title.md` 형식으로 추가합니다.

```markdown
---
title: "글 제목"
date: 2026-05-03 22:00:00 +0900
categories:
  - TIL
tags:
  - JavaScript
  - Algorithm
---

본문을 작성합니다.
```

## 배포

`main` 브랜치에 push하면 GitHub Pages에서 빌드됩니다.
